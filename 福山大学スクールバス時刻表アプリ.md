# 福山大学スクールバス時刻表アプリ

## 技術スタック
- React 18(TypeScript)
- TailWindCSS 3
- Vite
- React Router DOM(Routing)

## プロジェクト構造
```
game-info-site/
├── src/
│   ├── components/          # 再利用可能なUIコンポーネント
│   │   ├── TimeCard.tsx     # 時刻カード表示
│   │   ├── TimeList.tsx     # 時刻リスト表示
│   │   ├── TimeFilter.tsx   # フィルター機能
│   │   └── SearchBar.tsx    # 検索機能
│   ├── pages/               # ページコンポーネント
│   │   ├── Home.tsx         # トップページ
│   │   └── GameDetailPage.tsx # ゲーム詳細ページ
│   ├── hooks/
│   │   └── useGames.ts      # ゲームデータ管理フック
│   ├── types/
│   │   └── game.ts          # 型定義
│   ├── data/
│   │   └── games.json       # ゲームデータ
│   ├── utils/
│   │   └── dateUtils.ts     # 日付計算ユーティリティ
│   ├── App.tsx              # ルーティング設定
│   ├── main.tsx             # エントリーポイント
│   └── index.css            # グローバルCSS
├── tailwind.config.js
├── postcss.config.js
└── package.json
```
